# Hangman
Welcome to my Hangman Game! <br/>
In this game it is your job to guess the mystery word by guessing whether or not certain letters will be in the word. <br/>
However, beaware that losing too many lives would mean that you will be sent to the chopping block and lose the Game! <br/>
You are able to decide what category the mystery word can relate to but after that it is all up to you to figure out the word!<br/>
GOOD LUCK!
